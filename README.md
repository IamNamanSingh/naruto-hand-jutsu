# 🔥 Naruto Jutsu Hand Gesture Recognizer

A fun real-time hand gesture recognition project that lets you perform **Naruto-style jutsus** using your webcam!

Made with **OpenCV + MediaPipe** + cool particle effects and lightning.

## Features
- Real-time hand landmark detection
- Combo-based jutsus (like real Naruto hand signs)
- Dramatic visual effects:
  - **Rasengan** → Big glowing blue spinning chakra ball
  - **Chidori** → Electric lightning bolts + sparks
  - **Shadow Clone** → Multiple clone auras
  - **Fire Style: Fireball** → Flaming particles
- On-screen instruction tab showing exact combos
- Optional sound effects + screen shake
- Press `q` to quit

## Combo Guide (Perform quickly)
- **Shadow Clone Jutsu** → Open Hand → Ram Seal (Index + Middle fingers close together)
- **Rasengan** → Fist → Open Palm
- **Fire Style: Fireball Jutsu** → Pointing Finger → Ram Seal
- **Chidori** → Open Hand → Fist → Two Fingers
